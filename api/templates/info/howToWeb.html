How To: Website

1. Go to this link to see the website. You will be greeted by a homepage where you can log in or create an account.

2. Click on create an account.

3. Enter in a username and password.

4. You will be sent back to the homepage. Please log in using the same username and password you just created.

5. Your username should pop up next to a logout button in the top right corner. You will be sent to the events page where you can see all listed events. You will notice 4 tabs in the navigation bar, Home, Info, Events, and My Events. You can only access Home, Info, and Events when signed out.
